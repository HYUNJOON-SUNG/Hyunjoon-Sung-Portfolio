# Hyunjoon Sung Portfolio

Hyunjoon Sung의 포트폴리오 웹사이트입니다.

## 프로젝트 개요
이 프로젝트는 저의 기술 스택, 소속 활동, 주요 팀 및 개인 프로젝트 경험을 시각적으로 명확하고 현대적인 디자인으로 소개하기 위해 제작되었습니다. 

## 주요 내용
- **Hero Section**: 백엔드 및 클라우드 개발자로서의 지향점
- **Affiliations**: 주요 소속 및 활동 이력
- **Tech Stack**: Backend, Infrastructure, Tools 분야의 기술 역량
- **Projects**: 진행한 다양한 개발 프로젝트 포트폴리오

## 사용된 기술
- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion
